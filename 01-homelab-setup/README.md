# Home Lab Environment Setup

## Objective
Built a virtualized cybersecurity home lab to simulate real-world
attack and defense scenarios for SOC analyst skill development.

## Environment
- Host Machine: Windows 11
- Hypervisor: Oracle VirtualBox
- Virtual Machines:
  - Ubuntu 22.04 LTS — SIEM and log analysis server
  - Windows 11 Enterprise — target and victim machine
  - Kali Linux — penetration testing and attack simulation

## What I Built
- Configured a virtualized network with three VMs
- Allocated compute resources per machine role
- Established attacker, victim, and analyst machine roles
  mirroring a real SOC environment

## Skills Demonstrated
- Hypervisor configuration and VM deployment
- Virtual network architecture
- Linux and Windows system administration fundamentals
